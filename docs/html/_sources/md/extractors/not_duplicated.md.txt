# Filter duplicates
Some spiders may crawl duplicate data, such as `Tx Spider`. 
The built-in extractor of this project realizes the function of filtering duplicate transaction data.
You can run on this command in console:
```shell
python extract.py \
-i /path/of/raw/data \
-o /path/of/output/data
-e NotDuplicatedTxExtractor
-p True
```

Parameter Description:
- `-i`: the input directory of raw data.
- `-o`: the output directory of filtered data.
- `-e`: `Extractor` name.
- `-p`: Whether to enable parallel acceleration, the default is `False`.
